# Laboratório Web

Este repositório reúne os materiais, exemplos e atividades desenvolvidos durante as aulas de **Laboratório Web**.

Ao longo da disciplina, serão estudados conceitos e práticas relacionadas ao desenvolvimento web, incluindo:

* **Node.js** para desenvolvimento do lado do servidor (Back-end);
* Construção de interfaces e aplicações do lado do cliente (Front-end);
* Estruturação de projetos web;
* Manipulação de dados e integração entre cliente e servidor;
* Boas práticas de desenvolvimento e organização de código.

O objetivo é proporcionar uma base sólida para o desenvolvimento de aplicações web modernas, explorando tecnologias utilizadas no mercado e aplicando os conhecimentos em atividades práticas.

## Tecnologias abordadas

* JavaScript
* Node.js
* HTML
* CSS
* Desenvolvimento Front-end
* Desenvolvimento Back-end
